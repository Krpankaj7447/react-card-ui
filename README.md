# React Gallery App

This is a simple React application where data is fetched from an API and displayed on the screen.

## 🚀 Features
- Fetch data from API using Axios
- Display data dynamically using map()
- Use of React Hooks (useState, useEffect)
- Clean and simple UI

## 🛠️ Technologies Used
- React JS
- Axios
- JavaScript
- CSS / Tailwind (if used)

## 📡 API Used
- JSON Placeholder / Picsum Photos API

## 📚 Concepts Used
- useState Hook
- useEffect Hook
- API Calling
- Asynchronous JavaScript (async/await)
- Rendering lists using map()
